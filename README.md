# first-repo

This short notebook was a descriptive data analysis of an exam score dataset from 1000  high school students. Please also cite this dataset if you are to use it for your own analysis projects. 

Dataset: Chauhan, A. *Students Performance in Exams*. Kaggle, 2026

This analysis was purely exploratory. It tackled studying the distribution of the main quantitative variables, the test scores. It also went into going through the frequency breakdown of certain categorical variables. No certain statistical question, objective or hypothesis was in mind. 

Future iterations of the analysis should and will include inferential methods to further quantify observations in the dataset (one such example is the differences in test scores in the sample). More groupings and aggregations can also be analyzed using descriptive and inferential methods, which were pretty limited in this iteration.
